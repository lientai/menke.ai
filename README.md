# AI游戏生成器 🎮

一个基于Web的AI辅助游戏开发平台，提供可视化的游戏创建工具和智能聊天助手。

## 📋 项目概述

AI游戏生成器是一个创新的Web应用程序，旨在帮助用户通过拖拽操作和AI对话来快速创建网页游戏。该平台采用三栏布局设计，提供资源管理、游戏预览、属性编辑和AI助手等功能。

## ✨ 核心功能

### 🎯 资源管理系统
- **代码资源库**: 管理JavaScript游戏脚本文件
- **图片资源库**: 存储角色、敌人、背景、道具等图像资源
- **音乐资源库**: 管理背景音乐和音效文件
- **建模资源库**: 3D模型和几何体资源

### 🎮 游戏开发工具
- **可视化编辑器**: 拖拽式游戏对象创建
- **实时预览**: Canvas画布实时显示游戏效果
- **属性编辑器**: 精确调整位置、大小、旋转、透明度等属性
- **游戏控制**: 播放、暂停、停止游戏功能

### 🤖 AI智能助手
- **智能对话**: 理解自然语言需求并提供解决方案
- **快捷操作**: 一键创建游戏元素
- **实时帮助**: 根据用户操作提供相关建议
- **代码生成**: 自动生成游戏逻辑代码

## 🏗️ 技术架构

### 前端技术栈
- **HTML5**: 语义化标记和Canvas 2D绘图
- **CSS3**: 现代化样式、渐变效果、动画过渡
- **JavaScript ES6+**: 交互逻辑、事件处理、DOM操作
- **Font Awesome**: 图标库

### 设计特色
- **响应式布局**: 适配不同屏幕尺寸
- **毛玻璃效果**: 使用backdrop-filter实现现代化视觉效果
- **渐变背景**: 多彩渐变色彩方案
- **流畅动画**: CSS3过渡和变换效果

## 🚀 使用指南

### 快速开始
1. **克隆项目**: 下载所有文件到本地目录
2. **打开页面**: 使用现代浏览器打开 `index.html`
3. **开始创建**: 从左侧拖拽资源到中间画布

### 操作流程
1. **选择资源类型**: 点击左侧标签切换资源库
2. **拖拽创建对象**: 将资源拖拽到游戏画布
3. **编辑属性**: 使用下方编辑器调整对象属性
4. **AI辅助**: 与右侧AI助手交流获取帮助
5. **预览测试**: 使用控制按钮预览游戏效果

### 快捷键
- `Delete`: 删除选中的游戏对象
- `Ctrl+S`: 保存游戏项目
- `Ctrl+Z`: 撤销操作 (待实现)
